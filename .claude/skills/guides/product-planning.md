### 機能優先順位付け（RICE）

```typescript
interface Feature {
  name: string;
  reach: number;      // 影響を受けるユーザー数 (1-10)
  impact: number;     // インパクト (1-10)
  confidence: number; // 確信度 (0.5-1.0)
  effort: number;     // 工数（人週）
}

function calculateRICE(feature: Feature): number {
  return (feature.reach * feature.impact * feature.confidence) / feature.effort;
}

// 例
const features: Feature[] = [
  { name: 'ユーザー認証', reach: 10, impact: 8, confidence: 0.9, effort: 2 },
  { name: 'ダッシュボード', reach: 8, impact: 6, confidence: 0.8, effort: 3 },
  { name: 'API連携', reach: 5, impact: 7, confidence: 0.6, effort: 4 },
];

// RICEスコアでソート
const prioritized = features
  .map(f => ({ ...f, score: calculateRICE(f) }))
  .sort((a, b) => b.score - a.score);
```

## フレームワーク

### Jobs to be Done

```markdown
**When** [状況]
**I want to** [モチベーション]
**So I can** [期待する結果]

例:
When 朝の通勤時間に
I want to 最新のニュースをサッと確認したい
So I can 同僚との会話についていける
```
